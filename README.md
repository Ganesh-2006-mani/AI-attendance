# 🎓 AI Attendance System

This project predicts attendance and gives smart decisions using AI.

## Features
- History tracking (LocalStorage)
- Attendance trend graph (Chart.js)
- Pattern detection
- ML-based prediction
- Smart decision system

## Tech Stack
- HTML, CSS, JavaScript
- Python (Flask)
- Scikit-learn

## Run

1. Install dependencies:
   pip install flask scikit-learn numpy

2. Run backend:
   python app.py

3. Open index.html in browser
